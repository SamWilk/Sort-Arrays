This is a visualizer for a few sorting algorithms, created by Sam Wilk.
I will be adding more sorting algorithms soon, I used typescript and next js for this project.
There is a few visual bugs still when animating the sorting, will work on fixing those inbetween class and current internships
To operate the application, you will need to
  • Click the desired sorting algorithm
  • Once done animating click reset to randomize the array
  • Repeat
